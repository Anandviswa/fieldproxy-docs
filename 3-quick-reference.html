Top 5 Web Rules

  1. Every element needs a unique element_id
  2. space_widget in a row needs alignment: "horizontal"
  3. Use border_color + border_width — NOT CSS border: "1px solid ..."
  4. row and column ignore their own width — wrap them in a container
  5. onclick keys are numeric strings: "1", "2", "3" — never "1a", "1b"

Top 5 Mobile Rules

  1. Row onclick doesn't work — wrap in container
  2. expanded: true only on text — use it for all dynamic content in rows
  3. Badge containers need main_axis_size: "min" wrapper in rows
  4. Use navigation + back: true to close popups — not close_popup
  5. color: "transparent" doesn't work — just omit the color property

Top 10 Repeated Mistakes

  1. CSS border doesn't work — use border_color + border_width
  2. icon_button uses bg_color/color/size — NOT background_color/icon_color
  3. input_type: "search" doesn't exist — use "text"
  4. visibility_condition on inputs is ignored — wrap in container
  5. expanded: true only works on text — not containers/buttons
  6. Container bg uses color — NOT background_color
  7. show_if doesn't exist — use visibility_condition
  8. fixed_height doesn't exist — use height
  9. type: "divider" doesn't render — use container with height:1
  10. input_data keyed by element_id — NEVER by field_name

Verification Checklist (Run Before Shipping)

Web

  • Every element has unique element_id
  • No CSS border — uses border_color + border_width
  • expanded: true only on text
  • space_widget in rows has alignment: "horizontal"
  • Container backgrounds use color, not background_color
  • Data queries are SELECT-only (mutations in execute_query)
  • Popup grid widths scaled for popup size
  • grouped_counter_1 has all 5 required props

Mobile

  • Colored badges wrapped with main_axis_size: "min" parent
  • Dynamic text has expanded: true
  • Dividers use container + text child pattern
  • Tappable rows wrapped in container with onclick
  • Popup close uses navigation + back: true
  • No color: "transparent" (omit property instead)
  • No expanded: true on containers or buttons
